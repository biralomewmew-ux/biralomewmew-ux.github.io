# सार्वजनिक सेवा सन्तुष्टि सर्वेक्षण - सेटअप गाइड

यो प्रोजेक्टले फारम डाटा Google Sheets मा Apps Script मार्फत सेभ गर्छ र आकर्षक ड्यासबोर्डमा डाटा हेर्न सकिन्छ।

## फाइलहरू

- `index.html` - मुख्य सर्वेक्षण फारम
- `dashboard.html` - ड्यासबोर्ड पृष्ठ
- `dashboard.css` - ड्यासबोर्ड स्टाइलिङ
- `dashboard.js` - ड्यासबोर्ड JavaScript
- `code.gs` - Google Apps Script कोड

## सेटअप निर्देशन

### १. Google Sheets र Apps Script सेटअप गर्नुहोस्

1. **Google Sheets सिर्जना गर्नुहोस्**
   - [sheets.google.com](https://sheets.google.com) मा जानुहोस्
   - नयाँ स्प्रेडसिट सिर्जना गर्नुहोस्

2. **Apps Script खोल्नुहोस्**
   - स्प्रेडसिटमा `Extensions` > `Apps Script` मा क्लिक गर्नुहोस्
   - `code.gs` फाइलमा यो प्रोजेक्टको `code.gs` कोड कपी गर्नुहोस्

3. **Web App Deploy गर्नुहोस्**
   - Apps Script मा `Deploy` > `New deployment` क्लिक गर्नुहोस्
   - Select type: `Web app`
   - Description: "Customer Survey API"
   - Execute as: `Me`
   - Who has access: `Anyone` (वा `Anyone with Google account`)
   - `Deploy` क्लिक गर्नुहोस्
   - Web App URL कपी गर्नुहोस्

### २. Apps Script URL कन्फिगर गर्नुहोस्

**index.html मा:**
```javascript
const APPS_SCRIPT_URL = 'YOUR_APPS_SCRIPT_WEB_APP_URL_HERE';
```
लाइन ११३४ मा आफ्नो Web App URL राख्नुहोस्।

**dashboard.js मा:**
```javascript
const APPS_SCRIPT_URL = 'YOUR_APPS_SCRIPT_WEB_APP_URL_HERE';
```
लाइन ३ मा आफ्नो Web App URL राख्नुहोस्।

### ३. फाइलहरू होस्ट गर्नुहोस्

यी फाइलहरूलाई कुनै वेब सर्भरमा होस्ट गर्नुहोस्:
- GitHub Pages
- Netlify
- Vercel
- वा कुनै अन्य स्थिर वेब होस्टिङ

## प्रयोग गर्ने तरिका

### फारम भर्ने
१. `index.html` खोल्नुहोस्
२. आवश्यक जानकारी भर्नुहोस् (जिल्ला, उमेर समूह, मिति)
३. कार्यालयहरू खोल्नुहोस् र प्रश्नहरूको जवाफ दिनुहोस्
४. `फारम पेश गर्नुहोस्` बटन क्लिक गर्नुहोस्
५. डाटा Google Sheets मा सेभ हुनेछ

### ड्यासबोर्ड हेर्ने
१. `index.html` को header मा दायाँ माथि रहेको dashboard icon क्लिक गर्नुहोस्
२. वा सिधै `dashboard.html` खोल्नुहोस्
३. ड्यासबोर्डमा तपाईंले देख्न सक्नुहुन्छ:
   - कुल प्रतिक्रियाहरू
   - कार्यालय अनुसार प्रतिक्रिया
   - सन्तुष्टि स्कोर
   - उमेर समूह वितरण
   - जिल्ला अनुसार प्रतिक्रिया
   - विस्तृत डाटा टेबल

### फिल्टर प्रयोग गर्ने
- कार्यालय, जिल्ला, उमेर समूह, र मिति अनुसार फिल्टर गर्न सकिन्छ
- `रिसेट गर्नुहोस्` बटनले सबै फिल्टर हटाउँछ

### डाटा निर्यात गर्ने
- ड्यासबोर्डमा `निर्यात गर्नुहोस्` बटन क्लिक गरेर CSV फाइल डाउनलोड गर्न सकिन्छ

- ✅ इन्टरएक्टिभ चार्ट (Chart.js)

## समस्या समाधान

**डाटा सेभ हुँदैन:**
- Apps Script URL सही छ कि जाँच गर्नुहोस्
- Web App को access "Anyone" मा सेट छ कि जाँच गर्नुहोस्
- Console मा error हेर्नुहोस्

**ड्यासबोर्ड डाटा लोड हुँदैन:**
- Apps Script URL सही छ कि जाँच गर्नुहोस्
- Google Sheets मा डाटा छ कि जाँच गर्नुहोस्
- CORS सम्बन्धी समस्या हुन सक्छ
